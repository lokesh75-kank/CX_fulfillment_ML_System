# CX-Fulfillment Agent Frontend

Next.js frontend for the CX-Fulfillment Agent.

## Setup

```bash
npm install
npm run dev
```

## Pages

- `/` - Dashboard
- `/incidents/[id]` - Incident detail
- `/rca/[id]` - RCA report
- `/recommendations/[id]` - Recommendations

## API

Backend API runs on `http://localhost:8000`
